# EduPlatform

A Full Stack Learning Management System (LMS) built using Node.js, Express.js, MySQL, HTML, CSS, and JavaScript.

# Features

- User Authentication
- Student Dashboard
- Teacher Dashboard
- Admin Panel
- Course Management
- Mock Tests
- Payment Integration
- Responsive UI
- Secure Backend APIs

# Tech Stack

## Frontend
- HTML
- CSS
- JavaScript
- Bootstrap

## Backend
- Node.js
- Express.js

## Database
- MySQL
- Sequelize ORM
